数据分析第六步--撰写报告

发表于 2019-05-23 更新于 2019-11-29 分类于 数据分析 ， 基础理论 阅读次数： Valine：
本文字数： 3.3k 阅读时长 ≈ 6 分钟

# 数据报告

  数据分析报告是根据分析原理和方法，运用数据来反映、研究和分析某项事物的现状、问题、原因、本质和规律，并得出结论，提出解决办法的一种分析应用文体。这种文体是决策者认识事物、了解事物、掌握信息、搜集相关信息的主要工具之一，数据分析报告通过对事物数据全方位的科学分析来评估其环境及发展及环境发展状况，为决策提供科学、严谨的依据，降低风险。

  一份完整的数据分析报告，应当围绕目标确定范围，遵循一定的前提和原则，系统的反映存在的问题及原因，从而进一步找出解决问题的方法

# 数据报告的写作原则

  • 规范性
    • 数据分析中报告中所使用的名词术语一定要规范，标准统一，前后一致，要与业内公认的术语一致
  • 重要性
    • 数据分析报告一定要体现数据分析的重点，在各项数据分析中，应该重点选取关键指标，科学专业的进行分析。此外，针对统一类问题，其分析结果也应该按照问题重要性的高低来分级阐述。
  • 谨慎性
    • 数据分析报告的编制一定要谨慎，基础数据一定要真实、完整，分析过程必须科学、合理、全面，分析结果要可靠，内容要实事求是。
  • 创新性

# 数据分析报告的作用

  • 展现分析结果
  • 验证分析质量
  • 提供决策参考

# 数据报告的种类

# 专题分析报告

  专题分析报告是对社会经济现象某一方面后某一个问题进行专门研究的一种数据分析报告，他的主要作用是为决策者指定某项政策、解决某个问题提供决策参考和依据。专题分析报告主要有以下两个特点：

  • 单一性：专题分析报告不是要求反映事物的全貌，主要针对某一方面或某一问题进行分析，如用户流失分析、提升用户消费分析、提升企业利润率分析等。
  • 深入性：由于专题分析报告内容单一，重点突出，因此便于集中精力抓住主要问题的进行深入分析。它不仅要对问题进行具体的描述，还要对引起问题的原因进行分析，并且提出切实可行的解决办法。这就要对公司业务的认识要有一定的深度，由感性上升至理性，切记蜻蜓点水，泛泛而谈。

# 综合分析报告

  综合分析报告是全面评价一个地区，单位，部门业务或其他方面发展情况的一种数据分析报告。例如世界人口发展报告、全国经济发展报告、企业运营分析报告等。综合分析报告句意欧以下两个特点：

  • 全面性：综合分析报告反映的对象，无论一个地区，一个部门还是一个单位，都必须以这个单位为分析总体，站在全局的高度，反映总体特征，做出总体评价，得出总体认识。例如在分析方法论中提到的4p分析法，就是从产品、价格、渠道、促销四个角度进行企业分析。
  • 联系性：综合分析报告要把互联关联的一些现象、问题综合起来进行全面系统的分析。要在系统分析指标体系的基础上，考察对象之间的内部联系和外部联系其比例是否协调，适应。

# 日常数据通报

  日常数据通报以定期数据分析表为依据，反映计划的执行情况，并分析其影响和形成的原因的一种数据分析报告。这种数据分析报告一般是按日、周、月、季、年等时间阶段定期进行，所以也叫定期分析报告。一般包含以下3个特点：

  • 进度性：日常数据通报主要反映计划的执行情况，因此必须把计划执行的进度与时间的进展结合起来，观察两者的一致性，从而判断计划完成的好坏。需要进行一些必要的计算，通过一些绝对数相对指标来突出进度。
  • 规范性：日常数据通报基本上是数据分析部门的例行报告，定时向决策者提供。所以这种数据报告形成了比较规范的结构形式。一般有以下几个部分组成：
    • 反映计划执行的基本情况
    • 分析完成或未完成的原因
    • 总结计划执行中的成绩和经验，找出存在的问题
    • 提出措施和建议
  • 时效性：由日常数据通报的性质和任务决定，只有及时提供业务发展中的各种信息，才能帮助决策者掌握企业经营的主动权。

# 数据分析报告的结构

  数据分析报告主要采用“总——分——总”结构，主要包括：开篇、正文、结尾三大部分

# 标题页

  标题页需要写明报告的题目，题目要精简干练

标题常用的类型：

  1. 解释基本观点：用观点句来表示，点名数据分析报告的基本观点，如《不可忽视高价值客户保有》、《语音业务是公司发展的重要支柱》
  2. 概括主要内容：叙述数据反映的基本事实，概括分析报告的主要内容，让读者抓住全文的中心，如《2015年业务运行良好》、《我司销售额同比去年增长30%》
  3. 交代分析主题：反映分析的对象、时间、范围、内容等情况，并不点名主张和看法，如《2010年运营分析》，《2015年部门业务分析对比》
  4. 提出问题：以设问方式提出报告所需分析问题，引起读者思考，如《客户流失到哪里去了》、《公司收入下降的关键何在》《1500万元的利润从何而来》

标题制作要求：

  • 直接
  • 确切
  • 简洁：要直截了当的反映报告的基本内容和基本精神，要高度概括，用最少的文字准确简洁的表达
  • 艺术性：对外的分析报告切忌千篇一律，一定要有艺术性的表达

# 目录

  目录可以帮助读者了解报告的主要内容，并迅速找自己到所需内容。

# 前言

  前言是分析报告的一个重要组成部分，主要包括分析背景、目的及思路三方面，即：

  • 为何要展开此次分析，有何意义
  • 如何展开此次分析，主要通过哪几个方面
  • 如何展开此次分析，主要通过哪几个方面展开

# 分析背景

  对数据分析背景说明主要是让阅读者对整个分析研究的背景有所了解，主要阐述此项分析的主要原因、分析的意义，以及其他相关信息，如行业发展的现状等

# 分析目的

  数据分析报告中陈述分析目的的主要是为了让报告的阅读者了解开展此次分析能带来何种效果，可以解决什么问题。可以将分析的背景和目的同一起来

案例：

分析背景与目的

  • 经过近几年的发展，公司的客户规模达到XX万户，业务收入达xx万元，在诸多方面都取得了令人瞩目的业绩增长。。。。
  • 在业务增长的同时，行业竞争不断加剧、销售单价不断下跌，产品结构也日趋不合理
  • 期望通过对公司业务进行诊断分析，以及剖析以发现的问题，为明年的公司运营工作提供参考与指导，为取得新成绩打下坚实的基础。

# 分析思路

  分析思路用来指导数据分析师如何进行一个完整的数据分析，及确定需要分析的内容或指标。

# 正文

  正文是数据报告的核心部分。撰写正文时，根据之前的分析思路中确定的每项分析内容，利用各种数据分析方法，一步步展开分析通过图表及文字相结合的方式，形成报告正文。

   正文通过展开论题，对论点进行分析论证，表达报告撰写者的见解和研究成果的核心部分，因此正文占分析报告的绝大部分篇幅。一篇报告只有想法和主张是不行的，必须经过科学严密的论证才能确认观点的合理性和真实性。报告正文具有以下几个特点：

  • 是报告最长的主体部分。
  • 包含所有数据分析的事实和观点。
  • 通过数据图表和相关的文字分析结合分析。
  • 正文各部分具有逻辑关系。

# 结论和建议

  报告的结尾是对整个报告的总结、深化与提高，是得出结论、提出建议、解决关键矛盾的关键所在，它起着画龙点睛的作用，是整片分析报告的总结。

  结论是以数据分析为依据得出的分析结果，通常以综述性文字来说明。不是分析结果的简单重复，而是结合公司业务实际，通过综合分析、逻辑推理形成的总体论点。结论是去粗取精、由表里及里而抽象出的共同的、本质的规律，它与正文精密衔接，与前言相互对应，是分析报告首位呼应。结论应该措辞严谨、准确、鲜明。

  建议是根据数据分析结论对企业或业务等所面临的问题而提出的改进方法，建议主要关注在保持优势及改进劣势等方面。因为分析人员所出的建议主要是基于数据分析结果而得到的，会存在局限性，因此必须结合公司的具体业务才能得出切实可行的建议。

# 附录

  附录是数据报告的一个重要组成部分，它提供了正文中涉及而未予阐述的相关资料，有时也含有文中提及的资料，从而向读者提供一条深入数据分析报告的途径。它主要包括报告中涉及的专业名词解释、计算方法、重要的原始数据、地图等内容。每个内容都需要编号，以备查询。

# 数据报告的注意事项

  • 结构合理，逻辑清晰
  • 实事求是，反映真相
  • 用词准确，避免含糊
  • 篇幅适宜，简洁有效
  • 综合业务，分析合理
